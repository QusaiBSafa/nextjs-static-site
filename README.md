# Next.js Static Site

## Description
A simple static website built with Next.js utilizing static site generation for content.

## What's Built
- Home page
- About page
- Blog page with static content

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/nextjs-static-site.git
   cd nextjs-static-site
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and go to `http://localhost:3000`.

## API Docs
No APIs are exposed in this static site.

## Environment Variables
No environment variables are required.
